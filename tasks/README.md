# Задания для практики разрешения Git конфликтов

Этот проект содержит 33 задания для практики разрешения конфликтов Git.

## Как использовать

1. Выберите файл задания (например, `task-product-list.md`)
2. Прочитайте описания Задания A и Задания B
3. Найдите коллегу для совместной работы
4. Один из вас выполняет Задание A, другой - Задание B
5. Следуйте инструкциям Git в каждом задании
6. Попытайтесь смержить ветки и разрешить конфликты вместе!

## Типы конфликтов

- **Выбрать одно значение**: Выберите лучший вариант
- **Сохранить оба**: Объедините оба изменения
- **Реструктурировать**: Решите, как комбинировать логику

## Формат веток

Все ветки должны следовать формату: `ваш-никнейм/название-задания/a` или `ваш-никнейм/название-задания/b`

Пример: `ivan/product-list/a` или `maria/shopping-cart/b`

Удачи! 🚀
