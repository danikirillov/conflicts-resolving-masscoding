# Задание: Конфигурация портов Docker Compose

**Файл:** `devops/docker-compose.yml`

## Тип конфликта: Выбрать одно значение

### Задание A: Изменить веб-порт на 8080
**Расчетное время:** 5 минут

Обновите маппинг портов веб-приложения для использования 8080.

**Необходимые изменения:**
- Строка 11: Изменить `- "3000:3000"` на `- "8080:3000"`

**Обоснование:** Порт 3000 конфликтует с другими локальными сервисами.

**Инструкции по выполнению:**
```bash
git checkout -b yourname/docker-compose/a
git add devops/docker-compose.yml
git commit -m "Изменить веб-порт на 8080"
git push origin yourname/docker-compose/a
```

---

### Задание B: Изменить веб-порт на 4000
**Расчетное время:** 5 минут

Обновите маппинг портов веб-приложения для использования 4000.

**Необходимые изменения:**
- Строка 11: Изменить `- "3000:3000"` на `- "4000:3000"`

**Обоснование:** Стандартизировать все проекты для использования портов 4000+.

**Инструкции по выполнению:**
```bash
git checkout -b yourname/docker-compose/b
git add devops/docker-compose.yml
git commit -m "Изменить веб-порт на 4000"
git push origin yourname/docker-compose/b
```

---

## Ожидаемый конфликт
Оба задания изменяют маппинг внешнего порта. Может быть активен только один.

## Стратегия разрешения
**Выберите одно** на основе:
- Доступных портов на хост-машине
- Стандартов распределения портов в команде
- Конфигурации обратного прокси
