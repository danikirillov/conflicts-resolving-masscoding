# Задание: Пул подключений базы данных в конфигурации окружения

**Файл:** `devops/environment-config.json`

## Тип конфликта: Сохранить оба

### Задание A: Добавить настройку тайм-аута подключения
**Расчетное время:** 7 минут

Добавьте конфигурацию тайм-аута подключения к настройкам продакшн базы данных.

**Необходимые изменения:**
- После строки 73 (после `"pool_size": 50,`), добавить:
```json
        "connection_timeout": 30000,
```

**Обоснование:** Предотвратить зависшие подключения в продакшене.

**Инструкции по выполнению:**
```bash
git checkout -b yourname/environment-config/a
git add devops/environment-config.json
git commit -m "Добавить настройку тайм-аута подключения"
git push origin yourname/environment-config/a
```

---

### Задание B: Добавить настройку тайм-аута запроса
**Расчетное время:** 7 минут

Добавьте конфигурацию тайм-аута запроса к настройкам продакшн базы данных.

**Необходимые изменения:**
- После строки 73 (после `"pool_size": 50,`), добавить:
```json
        "query_timeout": 15000,
```

**Обоснование:** Прерывать медленные запросы, которые влияют на производительность.

**Инструкции по выполнению:**
```bash
git checkout -b yourname/environment-config/b
git add devops/environment-config.json
git commit -m "Добавить настройку тайм-аута запроса"
git push origin yourname/environment-config/b
```

---

## Ожидаемый конфликт
Оба задания добавляют новые опции конфигурации базы данных. Обе настройки тайм-аута ценны.

## Стратегия разрешения
**Сохраните оба** параметра - они контролируют разные аспекты тайм-аута и должны сосуществовать.
