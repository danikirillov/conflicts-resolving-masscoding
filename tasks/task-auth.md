# Задание: Тайм-аут сессии аутентификации

**Файл:** `frontend/php/auth.php`

## Тип конфликта: Выбрать одно значение

### Задание A: Увеличить тайм-аут сессии для удобства
**Расчетное время:** 5 минут

Увеличьте тайм-аут сессии до 2 часов для лучшего пользовательского опыта.

**Необходимые изменения:**
- Изменить `private $sessionTimeout = 3600;` на `private $sessionTimeout = 7200;`

**Обоснование:** Пользователи жалуются на слишком частый выход из системы.

**Инструкции по выполнению:**
```bash
git checkout -b <твой ник или имя>/auth/a
git add frontend/php/auth.php
git commit -m "Увеличить тайм-аут сессии до 2 часов"
git push origin <твой ник или имя>/auth/a
```

---

### Задание B: Уменьшить тайм-аут сессии для безопасности
**Расчетное время:** 5 минут

Уменьшите тайм-аут сессии до 30 минут для повышения безопасности.

**Необходимые изменения:**
- Изменить `private $sessionTimeout = 3600;` на `private $sessionTimeout = 1800;`

**Обоснование:** Аудит безопасности рекомендует более короткие тайм-ауты сессий.

**Инструкции по выполнению:**
```bash
git checkout -b <твой ник или имя>/auth/b
git add frontend/php/auth.php
git commit -m "Уменьшить тайм-аут сессии до 30 минут"
git push origin <твой ник или имя>/auth/b
```

---

## Ожидаемый конфликт
Оба задания изменяют тайм-аут сессии с противоположными целями (удобство vs. безопасность).

## Стратегия разрешения
**Выберите одно** на основе:
- Требований безопасности
- Приоритетов пользовательского опыта
- Лучших практик отрасли
- Требований соответствия
