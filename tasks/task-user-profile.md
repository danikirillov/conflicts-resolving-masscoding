# Задание: Уведомления профиля пользователя

**Файл:** `frontend/javascript/user-profile.js`

## Тип конфликта: Сохранить оба

### Задание A: Добавить настройку push-уведомлений
**Расчетное время:** 6 минут

Добавьте чекбокс для настройки push-уведомлений.

**Необходимые изменения:**
- после чекбокса SMS-уведомлений), добавить:
```javascript
                <label>
                    <input type="checkbox" id="pushNotifications"> 
                    Получать push-уведомления
                </label>
```

**Обоснование:** Пользователи мобильного приложения хотят контролировать push-уведомления.

**Инструкции по выполнению:**
```bash
git checkout -b <твой ник или имя>/user-profile/a
# Внесите изменения
git add frontend/javascript/user-profile.js
git commit -m "Добавить настройку push-уведомлений"
git push origin <твой ник или имя>/user-profile/a
```

### Задание B: Добавить подписку на рассылку
**Расчетное время:** 6 минут

Добавьте чекбокс для подписки на рассылку.

**Необходимые изменения:**
- после чекбокса SMS-уведомлений), добавить:
```javascript
                <label>
                    <input type="checkbox" id="newsletter"> 
                    Подписаться на еженедельную рассылку
                </label>
```

**Обоснование:** Маркетинговая команда хочет опцию подписки на рассылку в профиле.

**Инструкции по выполнению:**
```bash
git checkout -b <твой ник или имя>/user-profile/b
# Внесите изменения
git add frontend/javascript/user-profile.js
git commit -m "Добавить подписку на рассылку"

```

## Ожидаемый конфликт
Оба задания добавляют новые опции настроек в одном месте. Обе функции независимы и полезны.

## Стратегия разрешения
**Сохраните оба** чекбокса - пользователи должны иметь возможность контролировать как push-уведомления, так и подписку на рассылку. Добавьте обе опции в раздел настроек.
