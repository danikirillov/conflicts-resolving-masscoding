# Задание: Порог низкого запаса в админ-панели

**Файл:** `frontend/php/admin-panel.php`

## Тип конфликта: Выбрать одно значение

### Задание A: Понизить порог низкого запаса
**Расчетное время:** 5 минут

Измените порог оповещения о низком запасе с 10 до 5 единиц.

**Необходимые изменения:**
- Изменить `WHERE stock < 10` на `WHERE stock < 5`

**Обоснование:** Уменьшить шум оповещений - отмечать только критически низкий запас.

**Инструкции по выполнению:**
```bash
git checkout -b <твой ник или имя>/admin-panel/a
git add frontend/php/admin-panel.php
git commit -m "Понизить порог низкого запаса до 5"
git push origin <твой ник или имя>/admin-panel/a
```

---

### Задание B: Повысить порог низкого запаса
**Расчетное время:** 5 минут

Измените порог оповещения о низком запасе с 10 до 15 единиц.

**Необходимые изменения:**
-  Изменить `WHERE stock < 10` на `WHERE stock < 15`

**Обоснование:** Получать более ранние предупреждения для предотвращения дефицита.

**Инструкции по выполнению:**
```bash
git checkout -b <твой ник или имя>/admin-panel/b
git add frontend/php/admin-panel.php
git commit -m "Повысить порог низкого запаса до 15"
git push origin <твой ник или имя>/admin-panel/b
```

---

## Ожидаемый конфликт
Оба задания изменяют порог низкого запаса с противоположными целями (меньше vs. больше оповещений).
