# Задание: Конфигурация списка товаров

**Файл:** `frontend/javascript/product-list.js`

## Тип конфликта: Выбрать одно значение

### Задание A: Изменить количество товаров на странице
**Расчетное время:** 5 минут

Измените количество товаров, отображаемых на странице, с 12 до 16.

**Необходимые изменения:**
-  Изменить `this.itemsPerPage = 12;` на `this.itemsPerPage = 16;`

**Обоснование:** Маркетинговая команда просит показывать больше товаров на странице, чтобы уменьшить прокрутку.

**Инструкции по выполнению:**
```bash
# Создайте новую ветку (замените '<твой ник или имя>' на ваш никнейм)
git checkout -b <твой ник или имя>/product-list/a

# Внесите изменения в файл
# Сохраните и закоммитьте
git add frontend/javascript/product-list.js
git commit -m "Изменить количество товаров на странице до 16"

# Запушьте ветку
git push origin <твой ник или имя>/product-list/a
```

### Задание B: Изменить количество товаров на странице
**Расчетное время:** 5 минут

Измените количество товаров, отображаемых на странице, с 12 до 20.

**Необходимые изменения:**

**Инструкции по выполнению:**

## Ожидаемый конфликт
Оба задания изменяют одну и ту же строку. Вам нужно выбрать, какое значение имеет больше смысла, или обсудить с командой оптимальное значение.

## Стратегия разрешения
**Выберите одно** из значений (16 или 20) на основе:
- Соображений пользовательского опыта
- Производительности загрузки страницы
- Консенсуса команды
